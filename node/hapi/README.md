Authentication e Authorization
Armazenamento em cache cliente e servidor (Lib Catbox)
Roteamento - routes
Validação - Campos de payloud (Lib Joi)
Cookies - TRaz essa opção
Logging - Métodos de log nativos

### Comparando Hapi e Express
O Hapi tem uma estrutura padrão
Desenv. mais esperientes preferem o Express
Express usa midlewares para acessar a pipeline
Hapi abstrai funções do Node
Hapi usa plugins para acessar os midlewares

Boom.badRequest(error) - Envia as mensagens de erro no padrão
